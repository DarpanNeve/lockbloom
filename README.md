## ğŸ›¡ï¸ Security

LockBloom uses **AES-256-GCM encryption** with keys stored in platform secure storage (Android Keystore/iOS Keychain). Passwords are encrypted client-side and never leave your device.

**Security Features:**
- 256-bit encryption keys
- Cryptographically secure random generation
- PBKDF2 key derivation (100,000 iterations)
- Memory protection
- Code obfuscation enabled
